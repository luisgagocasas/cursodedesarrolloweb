Hola en el curso podran aprender todas estas herramientas y muchas tecnicas al desarrollar un sitio web desde lo mas basico con las herramientas que usan los grandes desarrolladores de las grandes empresas.

1

Sublime Text es un editor de código, su interfaz es limpia e intuitiva y soporta el uso de Snippets, Plugins y sistemas de construcción de código.

Entre sus caracteristicas tenemos:

  • Minimapa
  • Multi Selección
  • Multi Cursor
  • Multi Layout
  • Soporte nativo para infinidad de lenguajes
  • Syntax Highlight configurable
  • Búsqueda Dinámica
  • Auto completado y marcado de llaves
  • Soporte de Snippets y Plugins
  • Acceso rápido a linea o archivo

Puede descargarlo desde sublimetext.com

2

Stylus, un pre-procesador de CSS que les hará la vida más feliz.

¿Que es un Pre-Procesador CSS?
Son herramientas indispensables que nos ayudan a optimizar y administrar fácilmente nuestro código css. Con estas herramientas podemos realizar nuestro trabajo de una manera más ordenada y más rápida, ahorrando tiempo y dando mejores resultados. Stylus necesita de node.js corriendo para poder funcionar por lo tanto el primer paso será instalar node.js, para ello ingresamos al sitio nodejs.org

3

Con la ayuda del Navegador Chrome usaremos su barra de herramientas para ver al detalle las propiedades del codigo html, css y veremos unos plugins que nos ayurá a maquetar nuestro proyecto.

4

Git es un software de control de versiones, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

Entre sus caracteristicas tenemos:

  • Los almacenes de información pueden publicarse por HTTP, SSH, etc
  • Gestión eficiente de proyectos grandes
  • Todas las versiones previas a un cambio determinado
  • Etc.

5

Es una muy buena idea usar Fonts para tus Iconos en la web, es ves de estar usando imágenes pesadas, que lo único que hacen es aumentar las peticiones HTTP y aumentar la carga al servidor. Sin embargo utilizar Fuentes (Fonts) se está convirtiendo en un método popular en la web, ya que estas No aumentan la carga al servidor y se reducen enormemente las peticiones HTTP.

6

Últimamente muchos sitios web están basando su diseño en Bootstrap, ¿Qué es Bootstrap? Pues en pocas palabras es el framework de Twitter que permite crear interfaces web con CSS y Javascript que adaptan la interfaz dependiendo del tamaño del dispositivo en el que se visualice de forma nativa, es decir, automáticamente se adapta al tamaño de un ordenador o de una Tablet.
